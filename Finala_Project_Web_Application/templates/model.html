Model and Dataset

Our Model

Base on our experiment, we choose to use LogisticRegression as our model. We used MBTI dataset from kaggle to train our model. In order to fit the format of the MBTI dataset, we need to filter punctuation, stopwords, and emoji in the original tweets Then implement lemmatization to convert tweets to the normal format (no state of affairs).
Data preprocessing and feature extraction are necessary for Twitter streaming data. Moreover, the samples in the dataset distribute unbalanced, and some of the features only contain a few records. Therefore, we use oversampling and undersampling to reconstruct the dataset to make it balanced. The over-sampling technique SMOTE is used to duplicate these results for a more balanced amount of positive results in training. Our Dataset

The MBTI personality dataset is provided by Kaggle. It contains approximately 106k records of preprocessed posts and their authors' personality types. Every post of the record is equal sized and has 500 words in length.
